basicLinuxAPI

이 저장소는 Linux System Programming를 공부하기 위한 목적으로 만들어 졌으며, Micheal Kerrisk의 책 "The Linux Programming Interface" 를 기반으로 하여 개인이 이해하는 쉬운 source code로 재구성하거나, 다시 coding 되었다.

기초적인 개념을 이해하기 위한 내용은 추가적인 문서(*.html)로 기록할 것이며, code의 이해를 돕기 위한 내용은 대부분 code상에 주석(/* */, //)으로 기록될 것이다.

This repository contains personal modified examples for studying the contents of 'TLPI'(The Linux Programming Interface by Micheal Kerrisk).
